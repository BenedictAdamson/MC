MC-auth is the authentication server of the Mission Command game. It is a [Keycloak](https://www.keycloak.org/) server.

MC-auth is a Docker image, based on the [`quay.io/keycloak/keycloak` image](https://quay.io/repository/keycloak/keycloak). A Docker image for MC-auth is available from the  public repository
[https://hub.docker.com/r/benedictadamson/mc-auth](https://hub.docker.com/r/benedictadamson/mc-auth).

## Usage
Use of the `mc-auth` image is similar to use of the `quay.io/keycloak/keycloak` image.
